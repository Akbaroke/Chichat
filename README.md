## Introduction

Chitchat is a realtime chat web application with the convenience and user experience that makes it easy to communicate directly.

<br/>

## Tech Stack

- Reactjs Vite
- Typescript
- Redux
- TailwindCss
- Firebase

<br/>

## Features

- Sending Message
- Read Status Message
- Hide Message
- Login Oauth With Google Account
